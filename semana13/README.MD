## INSTALAR PRISMA
```npm install @prisma/client prisma dotenv```

## INICIALIZA O PRISMA
```npx prisma init```

## GERA O BUILD DAS ENTIDADES
```npx prisma generate```

## CRIA A MIGRATION PARA CRIAR O BANCO DE DADOS E AS ENTIDADES
```npx prisma migrate dev --name create-database```
